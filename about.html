Opam Report 

Why Opam Report ? 

Opam is a source package manager. One of its goals is to be portable accross several OS's and several Linux distros.

The goal of opam-install-report is to report to the opam packagers whether their packages build correctly or not on other OS's and other Linux distros.

How to Contribute ? 

Run the bash script and return (or commit) the logs.
